Introducción

Esta asignatura proporciona una visión completa de cómo funcionan las redes de computadores, incluyendo protocolos, arquitectura de red, modelos OSI y TCP/IP, direccionamiento IP, y tecnologías modernas como redes inalámbricas y seguridad de red.

Temas Principales

  • Modelo OSI y TCP/IP
  • Direcciones IP y Subnetting
  • Protocolos de enrutamiento (RIP, OSPF, EIGRP)
  • Redes LAN, MAN y WAN
  • Seguridad en redes
  • Redes inalámbricas y móviles

Objetivos del Curso

  1. Comprender la estructura y funcionamiento de una red de datos.
  2. Configurar dispositivos de red como routers y switches.
  3. Diseñar y diagnosticar redes utilizando herramientas como Packet Tracer.
  4. Analizar protocolos y su funcionamiento.
